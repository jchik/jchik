View://TTC-NextBus
TTC NextBus

A responsive web application that uses nextBusAPI to display a prediction of the next bus to arrive.

The application reads JSON from nextBusAPI and displays the next approximate bus time to arrive at a route and stop location. You are able to pin a route/bus combination that will be stored using the browser's localStorage. Pinned buses will auto refresh every 10seconds. You can manually refresh and remove a bus from the lookup or pinned list.

- HTML5/CSS3/Media Queries/JavaScript/jQuery/AngularJS -
